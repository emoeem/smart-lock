**项目背景与描述：**  针对传统门禁系统认证方式单一、易受复杂光照/环境噪声干扰，且缺乏远程管控能力等行业痛点，自主设计并主导开发了一款基于“边缘计算 + 云端管理”架构的多模态智能门禁系统，打通了人脸、声纹、物理密码及微信小程序的多维融合认证闭环。

**核心职责与技术实现：**

- **AI 算法端侧部署与优化：**  针对复杂环境的识别难题，人脸识别模块引入 ArcFace 等多模型融合，并创新性加入**动态阈值**与**增量学习机制**，使系统能自适应用户的面部特征变化；声纹识别采用 CAM++ (D-TDNN) 模型，针对树莓派算力进行深度推理优化，实现了低延迟的本地特征提取与比对。
- **边缘计算与软硬协同：**  基于树莓派搭建边缘侧网关，使用 Python/FastAPI 与 `gpiozero` 库完成对电磁锁的底层物理控制，并实现了多线程的音视频流异步采集，保障系统响应的流畅性。
- **高并发云端架构设计：**  后端基于 Spring Boot + MySQL 构建业务中台与日志审计体系；为解决远程控制的高实时性需求，引入 **MQTT 协议**建立设备与云端的长连接，确保远程指令下发与状态同步达到毫秒级响应。
- **跨平台应用开发：**  使用 uni-app 独立开发微信小程序管理端，提供多设备动态绑定、实时开门日志查询、远程动态密码管理等功能，极大提升了系统的可维护性与用户体验。

**项目成果与业务价值：** 成功完成软硬件一体化的低成本落地验证。通过优化的存储结构设计，系统具备**离线断网秒级比对**与**联网日志自动同步**的高可用特性；多模态融合技术的应用，彻底规避了单一生物特征被伪造的风险，整体识别准确率与系统安全性得到显著提升。
